# Employee Salary Calculator

A simple C# console application that calculates an employee's salary including HRA, allowance, and tax deductions.

## Features
- Accepts employee name and basic salary
- Calculates HRA (20% of basic salary)
- Calculates allowance (10% of basic salary)
- Calculates tax deduction (5% of basic salary)
- Displays net salary breakdown

## Technologies Used
- C#
- .NET Console Application
